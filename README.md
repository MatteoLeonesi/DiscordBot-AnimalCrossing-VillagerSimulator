## About The Project
NOT-OFFICIAL bot created for fun and learning purposes, that simulates animal crossing new horizon villager. User can catch a fish, or bug or meeting villagers, and having a serverless inventory, it's still in development if you wanna contribute text me üåª !


### Built With
* [DiscordJs](https://discord.js.org/#/)
* [acnhapi](http://acnhapi.com/)
* [undici](https://github.com/nodejs/undici)

<!-- GETTING STARTED -->
## Getting Started
Start the server, after created config.json or env file with token,client id, guild id attributes : 
  ```sh
 npm i 
 nodemon 
  ```
